Win10使用VMware Workstation pro 15 安装Ubuntu Server18.04.2LTS

发表于 2019-03-08 | 更新于 2019-03-10 | 分类于 安装教程 | 评论数：

一、准备工作

1. 下载VMware workstation

在wmware官网上下载我们的安装包文件,打开按提示安装即可，具体的安装密钥可参照这篇文章VMware Workstation Pro15.0安装

2. 下载Ubuntu Server18.04.2LTS

进入Ubuntu官网下载Ubuntu Server

二、新建虚拟机

1. 自定义安装

2. 选择硬件兼容性

3. 选择稍后安装操作系统

4. 根据自己要按照的客户机版本选择，这里我们选择Ubuntu 64位

5. 定义虚拟机的名称及安装位置（可以根据自己需求爱好设置）

6. 处理器配置，一般默认，如果电脑配置好不妨增加处理器数量

7. 虚拟机内存，选择默认或者适当调大

8. 网络类型，默认

9. I/O控制类型，默认

10. 磁盘类型，默认

11. 磁盘选择创建新虚拟磁盘

12. 磁盘容量选择默认的20GB（可根据需要自行更改），这里我选择立即分配所有磁盘空间和存储为单个文件，可以降低虚拟机卡顿

13. 指定磁盘文件，默认

14. 镜像配置，选择下载好的镜像，完成


15. 等待磁盘创建

三、安装

1. 开启虚拟机

2. 选择下载，按Enter选择

3. 键盘配置，默认

4. 安装Ubuntu 按Enter选择

5. 网络连接，默认

6. 无需代理配置，默认

7. 镜像源选择，默认

8. 选择整个磁盘

9. 文件系统选择，默认



10. 配置文件，填写你的账户名和密码，不能为空

11. 等待安装完成，重启

12. 安装完成

-------------The End-------------
